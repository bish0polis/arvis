# `arvis-workflow-json` format

The json file consists of the following attributes.

## defaultIcon

type: `string`

required: `false`

## category

type: `string`

required: `false`

## createdby

type: `string`

required: `false`

## description

type: `string`

required: `false`

## name

type: `string`

required: `false`

## readme

type: `string`

required: `false`

## version

type: `string`

required: `false`

## webaddress

type: `string`

required: `false`

## enabled

type: `string`

required: `true`

## commands

type: `array`

required: `true`

commands consist of `trigger-action` pairs.
